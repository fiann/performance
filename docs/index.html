Current browser spec

Your device pixel ratio is: . Viewport width is (CSS pixels) or (physical pixels).

Resize your browser window to see which image variants are loaded. The browser will load a larger version of the image if you increase the size of the window. It will not load a smaller version of the image if you make the window smaller, it will simply scale down the existing image, unless you cross a Media Query breakpoint.
