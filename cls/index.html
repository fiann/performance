Cumulative Layout Shift with img element

This demo shows the effect of correctly setting image height to help reduce Cumulative Layout Shift. Use network throttling in the browser dev tools to see the effect.

Next page: Using the <picture> tag

Image with no dimensions:

Any content under the image is going to move, causing a Layout Shift.

With dimensions:

Image size reserves space, removing the Layout Shift.

HTML code including image height and width

<img class="full-width" id="hero-image"
  width="1024" height="530"
  sizes="50vw"
  srcset="
    https://www.clinique.com/media/export/cms/2101_assets/Sp21_EBCF_LMW_dt@2x.jpg?imWidth=414   414w,
    https://www.clinique.com/media/export/cms/2101_assets/Sp21_EBCF_LMW_dt@2x.jpg?imWidth=828   828w,
    https://www.clinique.com/media/export/cms/2101_assets/Sp21_EBCF_LMW_dt@2x.jpg?imWidth=1440 1440w,
    https://www.clinique.com/media/export/cms/2101_assets/Sp21_EBCF_LMW_dt@2x.jpg?imWidth=1920 1920w,
    https://www.clinique.com/media/export/cms/2101_assets/Sp21_EBCF_LMW_dt@2x.jpg?imWidth=2880 2880w
  "
  src="404.jpg"
/>

Note that the height and width of the final rendered image are specified from CSS. The width and height attributes of the <img> tag specify the aspect ratio of the image, not the rendered size.

Image height and width are handled automatically by the browser, with no additional javascript required on the page.
